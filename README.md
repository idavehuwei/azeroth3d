# 熔火之心 · Azeroth3D

**经典风格的 3D 网页游戏：无外部美术资源，所有模型/图标/特效程序化生成。**

一个极简的 MMO 风格战斗模拟，基于 Three.js 构建，所有 3D 模型、法术图标、粒子特效全部由代码在运行时生成，零外部图片、模型、音频文件。

> 在线体验：打开 `game.html`，选择职业，沿土路北行，踏入传送门，讨伐炎魔领主拉戈斯。

---

## 游戏概览

### 背景

牛头人长老在营火旁低语：北方矗立着一道燃烧的传送门，通往地底深处的熔火之心——炎魔领主拉戈斯正在那里积聚力量。选择你的职业，踏入旋涡，终结他。

### 职业

| 职业 | 类型 | 特点 |
|------|------|------|
| 战士 | 近战 | 怒气系统，冲锋/旋风斩/英勇打击，高生存 |
| 法师 | 远程法术 | 法力恢复，炎爆术/冰霜新星/闪现/寒冰屏障 |
| 弓箭手 | 远程物理 | 能量恢复，瞄准射击/多重射击/翻滚闪避 |

### 操作

- **WASD / 方向键**：移动
- **数字键 1-4**：释放技能
- **F**：对话 / 拾取
- **触屏**：摇杆控制移动，点击技能按钮

## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/idavehuwei/azeroth3d.git
cd azeroth3d
```

2. 启动游戏
```bash
# 直接打开 game.html（需要本地 HTTP 服务器以加载 Three.js CDN）
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080/game.html
```

3. 游戏流程
   - 选择职业 → 点击「启程」
   - 在莫高雷草原与长老对话，接受任务
   - 猎杀 3 只草原野猪
   - 回到长老处领取奖励，接受讨伐任务
   - 沿土路北行，踏入传送门
   - 击败炎魔领主拉戈斯

## 项目架构

```
js/
├─ core.js      基础设施：BALANCE 数值表 / SeededRng / 渲染器 / 场景 / makeLabel
├─ icons.js     图标画布工厂：8 个配方 + 画笔函数库（STEP 2）
├─ items.js     物品与掉落系统：品质表 / 掉落表 / 实体 / 拾取（STEP 2）
├─ models.js    3D 模型工厂：三职业 / Boss / 野猪 / 烈焰之子 / 长老
├─ world.js     莫高雷世界：草原 / 营地 / 传送门 / 野怪与 NPC 任务系统
├─ combat.js    战斗系统：游戏状态 / 职业 / 技能 / 统一受击 / Boss AI
└─ main.js      主循环与启动：每帧更新 / 相机 / UI 刷新 / 职业选择
```

### 核心设计原则

- **资源 = 工厂函数 + 数据参数**：所有美术/音频产物由可复用的工厂按数据参数生成，仓库中无 .png/.glb/.mp3 文件
- **Entity 统一受击**：野猪/烈焰之子/Boss 走同一个 `hitEntity` 入口，掉落与经验挂接在 `onDeath` 回调
- **SeededRng 确定性世界**：世界摆放类随机走可播种随机器，种子固定，世界一草一木位置不变
- **数值外置 BALANCE**：技能伤害、怪物血量、Boss 技能 CD 全部写在 `BALANCE` 常量表，改平衡不碰逻辑代码

## 技术栈

| 技术 | 用途 |
|------|------|
| Three.js r128 | 3D 渲染引擎（CDN 引入） |
| Canvas 2D | 运行时图标绘制（技能/物品图标） |
| WebGL | Shader 材质（熔岩动画、传送门旋涡） |
| 纯 DOM + CSS | UI 界面（零框架） |
| 纯 JavaScript | 全部游戏逻辑（ES6，无构建工具） |

## 参考与致谢

本项目参考了 [World of ClaudeCraft](https://github.com/levy-street/world-of-claudecraft)（MIT 许可）的以下设计：

- **One sim, three hosts** 架构思想（数值与渲染分离的雏形）
- **Content as data** 数据驱动设计（技能、掉落、怪物均以数据表定义）
- **尸体拾取**（先杀怪出尸体、按 F 拾取的手感）
- **程序化 WebAudio** 音效合成方案（音频零资源）
- **程序化图标** 的 Canvas 2D 绘制路线
